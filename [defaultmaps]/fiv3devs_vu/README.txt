- How do you change the "sexy rooms"?

VIDEO TUTORIAL: https://youtu.be/cbn6EEvcM_M

1) Open file entitysets.lua with your favorite text editing software
2) Replace DisableInteriorProp with EnableInteriorProp for the version you wish to see
3) Only ONE entityset per room should be enabled at one time, rooms are named after colors: red, pink, purple, blue
4) 1 = love bed
   2 = prison
   3 = bdsm
   4 = hospital
5) Save and restart the script
